1. Open the solution in visual studio.

2. Restore nuget packages from nuget.org package source. I have used RestSharp nuget package to make api call.

3. Build the solution.

4. Run the test cases.

5. Run the solution in browser. It will use IIS Express.

6. Click on Draw Results link to go to results page.

7. Select product by clicking the check box and click on display draw results.
